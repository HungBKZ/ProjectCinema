# Cinema Booking System

Hệ thống đặt vé xem phim cho phim "ĐỐI" tại Lotte Cinema Ninh Kiều - Cần Thơ

## Công nghệ sử dụng
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

## Cài đặt

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

- **Ngày chiếu**: 28/12/2024
- **Giờ chiếu**: 18h30

## Tính năng
- Đăng ký/Đăng nhập tài khoản
- Xem thông tin phim
- Chọn ghế (tối đa 5 ghế, không được bỏ lỗ)
- Giữ ghế 10 phút
- Thanh toán qua QR Code
- Admin xác nhận đơn hàng
